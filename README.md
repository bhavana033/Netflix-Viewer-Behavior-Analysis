## Netflix Viewer Behavior Analysis

Project Overview
This project analyzes Netflix viewer behavior using Python, SQL, and Power BI. It focuses on understanding viewer engagement, watch time, subscription plans, device usage, and content preferences through data analysis and interactive visualizations.

 Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (MySQL)
- Power BI

 Project Workflow
1. Cleaned and prepared the dataset using Python.
2. Performed SQL queries to analyze viewer behavior and generate business insights.
3. Built an interactive Power BI dashboard to visualize key metrics and trends.

📊 Dashboard Features
- KPI Cards
  - Total Users
  - Average Rating
  - Average Weekly Watch Time
  - Completion Rate
- Top 10 Countries by Users
- Device Type Distribution
- Subscription Plan Distribution
- Average Rating by Subscription Plan
- Average Watch Time Trend
- Average Watch Time by Genre
- Content Popularity by Genre
- Interactive Filters (Gender, Country, Subscription Plan, Device Type)

 🔍 Key Insights
